# BuilderBee Skills

Client brand, copy standards, and voice guidelines for Claude.

## How to use
When starting any client task, tell Claude:
> "Read the [client-name] skill before starting"

Claude will read the SKILL.md for that client and apply all brand,
copy, and voice rules automatically throughout the session.

## Clients
| Folder | Client |
|--------|--------|
| `alchemy-of-breath/` | Alchemy of Breath (AOB) |
| `yogahub/` | yogahub (multi-location yoga/pilates studios) |
| `bbtrs/` | BBTRS — BioDynamic Breathwork & Trauma Release System |
| `dr-lorenc/` | Dr. Lorenc Aesthetic Plastic Surgery Center (NYC) |
| `builderbee/` | BuilderBee universal standards (apply to all clients) |

## Structure
Each folder contains:
- `SKILL.md` — main file Claude reads before every task
- `README.md` — human-readable brand overview
